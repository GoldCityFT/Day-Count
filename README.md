# Day Counter PWA

เว็บ Day Counter ส่วนตัวสำหรับติดตั้งบนหน้าจอมือถือ ใช้งานผ่าน GitHub Pages ได้ฟรี

## จุดเด่น

- เลือกวันเริ่มต้นและชื่อที่ต้องการแสดงได้
- แสดงผลแบบ ปี / เดือน / วัน และจำนวนวันรวม
- มีเป้าหมายจำนวนวันถัดไป
- ข้อมูลเก็บใน Local Storage ของเครื่อง ไม่ส่งไปที่ GitHub
- เปิดใช้งานแบบออฟไลน์ได้
- ติดตั้งบนหน้าจอมือถือเหมือนแอปได้

## วิธีนำขึ้น GitHub Pages

1. สร้าง Repository ใหม่ใน GitHub เช่น `day-counter`
2. อัปโหลดไฟล์ทั้งหมดในโฟลเดอร์นี้ขึ้น Repository
3. เข้า `Settings` → `Pages`
4. ที่หัวข้อ `Build and deployment`
5. เลือก `Deploy from a branch`
6. เลือก Branch `main` และ Folder `/ (root)`
7. กด `Save`
8. รอสักครู่ แล้ว GitHub จะแสดงลิงก์เว็บไซต์

ลิงก์โดยทั่วไปจะเป็น:

`https://ชื่อผู้ใช้.github.io/day-counter/`

## เพิ่มไว้บนหน้าจอมือถือ

### Android / Chrome
เปิดเว็บไซต์ แล้วกดปุ่ม “เพิ่มไว้บนหน้าจอมือถือ” หรือเลือก Install app จากเมนู Chrome

### iPhone / Safari
เปิดเว็บไซต์ใน Safari → กด Share → Add to Home Screen

## แก้ไขภายหลัง

กดปุ่มรูปเฟืองมุมขวาบน เพื่อเปลี่ยนชื่อ วันที่เริ่มต้น หรือสีหลัก
